About the latest release

This release is based on Ruby 1.8.2 and as such the kit has not been fully tested.

Currently, some selected extension libraries are included. Many of the core ruby libraries (written in Ruby) are included; However, these libraries have not been thoroughly tested on OpenVMS.

Following extension libraries are included.

  • socket
  • digest
  • fcntl
  • nkf
  • sdbm
  • iconv (added in T1.8-2X014)
