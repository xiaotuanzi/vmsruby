Ruby for OpenVMS Release Summary

T1.8-2X012 (Latest version)

  • Released on December 17, 2004.
  • Based on Ruby 1.8.2 preview 3 (2004-12-16)
  • Some extended libraries have been added. (nkf, digest and digest/md5)
  • File#flock has been fixed. (Need OpenVMS V7.3-2 or V7.3-1 ACRTL ECO V3)
  • PStore can be used. (Please test it and report to me.)

T1.8-2X005

  • Released on December 9, 2004.
  • Based on Ruby 1.8.2 preview 3 (2004-12-09)
  • The first release based on Ruby 1.8.2.

X1.8-1X014

  • Release on August 9, 2004.
  • Based on Ruby 1.8.1 (2003-12-15)
  • The first release of Ruby for OpenVMS.
  • Socket extended library is included.
